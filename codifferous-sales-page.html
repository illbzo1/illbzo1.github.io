Codifferous Sales Page

https://codifferous.com/

copywriting, customer development, keyword research

This project was a second take on sales copy for Codifferous. The original version had good conversion rates, but people who signed up weren't reviewing code. I wanted to know why. I spent a week talking to 40 developers about code review, then rewrote the copy to position Codifferous as a solution for the biggest issues developers talked about - code review takes too much time, and it's too difficult.

The revised copy saw a large improvement in conversion and adoption rates, as well as long-term improvement in organic results for search terms related to code review.
